# Minimal TODO Example

This example demonstrates a minimal `tsk` workspace for a solo developer.
It includes a small TODO app with six tasks and simple dependencies.

## Layout
- `tasks/` is the required root for canonical task paths.
- All task references in front matter are relative to `tasks/`.

## Task Set
- `todo-app/initialize-project`
- `todo-app/add-item`
- `todo-app/complete-item`
- `todo-app/delete-item`
- `todo-app/persist-storage`
- `todo-app/filter-items`
